## Notable Absences

- **No build step**: pi loads `.ts` source directly. `tsconfig.json` has `noEmit: true`.
- **No runtime dependencies**: All provider logic is self-contained with Node.js built-ins (`fs`, `os`, `path`).
- **No framework**: This is a pi extension, not a standalone server.
